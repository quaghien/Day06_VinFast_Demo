# VinFast AI Demo

Dự án phát triển AI Agent cho VinFast dựa trên Streamlit.

## Cấu trúc dự án
(Sẽ được cập nhật)

## Yêu cầu
- Python 3.10+
- Các thư viện cần thiết

## Hướng dẫn chạy
1. Tạo và kích hoạt môi trường ảo (ví dụ: `python -m venv venv` và `source venv/bin/activate` hoặc `venv\Scripts\activate` trên Windows)
2. Cài đặt thư viện: `pip install -r requirements.txt` (khi đã có file yêu cầu)
3. Chạy ứng dụng Streamlit: `streamlit run app.py` (điều chỉnh tên file chính nếu cần)
